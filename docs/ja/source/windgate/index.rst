===============================
外部システムとの連携 - WindGate
===============================

WindGateはバッチに対するデータの外部入出力を行うモジュールです。

ThunderGateに対して、WindGateは「ポータブルなデータ転送ツール」して位置づけられ、以下の特徴を持ちます。

フラットファイルに対応
======================

ローカルファイルシステム上のフラットファイル(CSV形式)に対するデータ入出力に対応しています。

様々なRDBMSに対応
=================

DBMS固有の機能に依存せず、標準SQL/JDBCインターフェースのみを使用した実装を提供します。

また、処理対象テーブルに対するThunderGateの管理カラムの追加が不要です。
ただしThunderGateが提供していたロック機構などの仕組みが一部提供されません。

様々な構成に対応
================

様々なクラスタ構成やOS上で動作するように設計されています。

外部から拡張しやすい
====================

データ入出力部分を拡張しやすいアーキテクチャを採用しています。
