..  attention::
    試験的機能とは以下のような状態で公開されている機能を示します。
    
    * 十分な検証が行われていない
    * 限られたプラットフォーム上でのみ動作
    * 暫定仕様の状態で公開されている状態で、将来大きな仕様変更が行われる可能性がある
    * ドキュメントが未作成または十分ではない
    * 特定のプラットフォーム上の利用に関する使用例などの情報を提供するドキュメント
