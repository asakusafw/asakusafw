==========
試験的機能
==========

試験的機能とは、以下のいずれかに該当するような状態で公開している機能です。

* 十分な検証が行われていない
* 限られたプラットフォーム上でのみ動作する
* 暫定仕様の状態で公開されているため、将来大きな仕様変更が行われる可能性がある
* ドキュメントが未作成または十分ではない

また、特定プラットフォーム上での使用例などを紹介するドキュメントも、ここで公開しています。
